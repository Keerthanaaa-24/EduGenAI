# EduGen AI

AI-Powered Educational Assistant built using MERN Stack, Gemini AI, MongoDB, OCR and RAG concepts.

---

## Features

### Authentication

* Register
* Login
* JWT Authentication

### Document Processing

* PDF Upload
* Image Upload
* OCR Text Extraction
* PDF Text Extraction

### AI Features

* AI Chat with Notes
* Quiz Generation
* Notes Summary
* Study Planner

### Analytics

* Documents Uploaded
* Quizzes Taken
* Study Plans Created
* Average Quiz Score

---

## Tech Stack

### Frontend

* React
* React Router
* Axios
* Recharts

### Backend

* Node.js
* Express.js
* MongoDB
* Gemini API
* Tesseract OCR
* ChromaDB

## Installation

### Backend

cd server

npm install

npm run dev

### Frontend

cd client

npm install

npm run dev

---

## Environment Variables

PORT=5000

MONGO_URI=your_mongodb_connection

JWT_SECRET=your_secret

GEMINI_API_KEY=your_api_key

CHROMA_URL=http://localhost:8000
